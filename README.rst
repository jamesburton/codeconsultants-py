Install
------------------------------------------------------------------------------

``codeconsultants`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install codeconsultants

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade codeconsultants
